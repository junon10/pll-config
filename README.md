# PLL CONFIG

## Description

This app generates the combination of keys necessary to configure various types of PLLs

## Features

- **Compatibility**: Works with Junon's PLL, Pira's PLL, CD40103B+CD4046B CMOS AM PLL, etc.
- **multiplatform app**: Runs directly in the web browser with javascript support.

## Requirements

- **System**: Web browser with JavaScript support.

## Installation

To run the app (HTML + Javascript).

1. Just open the `index.html` file in your Web Browser. It is not necessary to compile, as it is an interpreted language.
